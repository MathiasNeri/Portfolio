# Portfolio - Mathias Neri

Portfolio personnel développé avec React et Tailwind CSS.

## 🌐 **Site en ligne**
**URL :** https://mathiasneri.github.io/Portefolio/

## 🚀 **Technologies utilisées**
- React 19
- Tailwind CSS
- Framer Motion
- EmailJS
- Vite

## 📦 **Installation et déploiement**

### Installation locale :
```bash
cd portfolio-ynov
npm install
npm run dev
```

### Déploiement sur GitHub Pages :
```bash
npm run deploy
```
